## About

This is the source code for my personal portfolio website. It's a single-page React application that showcases my projects, work experience, education, and skills. The site is fully client-side rendered and deployed on Vercel.

The design uses a dark aesthetic with a liquid-glass UI style, blur animations, and a full-screen video hero. Built with Tailwind CSS utility classes inlined into a single `index.html` output — no build step required on the client side.

## Site Structure

```
portfolio-site/
├── index.html          # Main entry — full React app compiled to a single file
├── vercel.json         # Vercel deployment config
├── sync-and-push.ps1   # PowerShell helper for syncing & pushing changes
└── README.md           # You're here
```

The site is structured as a single-page app with the following sections:

- **Hero** — Full-screen video background with name, title, and CTA
- **About** — Bio, skills, and stats
- **Projects** — Interactive project explorer with details
- **Experience** — Work history timeline
- **Education** — Academic background and courses
- **Contact** — Email, LinkedIn, GitHub links
- **Footer** — Navigation and social links

## Running Locally

Since this is a pre-compiled single-file React app, you can open it directly in a browser:

```bash
# Clone the repo
git clone https://github.com/zanni098/portfolio-site.git
cd portfolio-site

# Open in browser (no build step needed)
open index.html

# Or serve locally
npx serve .
```

---

## Deployment

The site is deployed on **Vercel** with the configuration in `vercel.json`. Any push to `main` triggers an automatic deployment.
